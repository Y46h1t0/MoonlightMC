# MoonlightMC

MoonlightMC is a general-purpose Minecraft utility mod specifically designed for competitive gameplay. The project operates as an addon for **Cactus Mod**, expanding its capabilities through a comprehensive suite of modular features.

All integrated features are built with strict adherence to **fair play**. The mod does not contain any form of unfair advantages or cheats, making it safe and compliant for use on competitive multiplayer servers.

## Key Features

* **Full Modularity**: Every feature can be individually enabled, disabled, and configured via the in-game GUI to suit user preferences.
* **Competitive Optimization**: Advanced tools tailored for performance monitoring, HUD customization, and real-time data management.
* **Native Integration**: Seamless synergy with the Cactus Mod architecture, ensuring stability and preventing module conflicts.
* **Game Integrity**: Strictly developed to exclude any malicious or rule-breaking mechanics, ensuring a clean competitive experience.

## Configuration

To access the configuration panel and customize MoonlightMC modules in-game:
* Press the default keybind: **RIGHT SHIFT**
* Use the graphical interface to toggle or adjust individual modules in real time.
